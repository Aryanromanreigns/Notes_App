📝 Flutter Notes App
A beginner-friendly, open-source Flutter application to create, store, and manage your notes efficiently. Built with clean architecture and modular code so contributors can easily understand and enhance the app.

📱 Features
➕ Add new notes with title & description

✏️ Edit existing notes anytime

🗑️ Delete notes with a single tap

💾 Offline storage using SQLite database

🌐 Responsive layout (works on both portrait and landscape)

🧼 Simple, clean, and readable Flutter code

👩‍💻 Beginner-friendly codebase for open-source contributors

🏁 Getting Started
Prerequisites
Flutter SDK: Install Flutter

Android Studio / VS Code with Flutter plugin
